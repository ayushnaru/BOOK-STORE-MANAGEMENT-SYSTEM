# BOOK-STORE-MANAGEMENT-SYSTEM

A demo web application for managing books, authors, and publishers, with password-protected "classified" book list and a spy-themed interface.

## Technologies
- Python 3 + Flask
- MySQL (with sample schema and data)
- HTML/CSS/JavaScript

## Features
- Add and list books, authors, and publishers.
- Book list hidden behind password prompt ("2530").
- Auto-create authors/publishers from book entry.
- Sample SQL and project report included.

## How to Run
1. Import SQL schema and sample data into your MySQL database.
2. Edit sap.py with your MySQL login details.
3. Run python sap.py.
4. Open in browser: http://localhost:5000/
